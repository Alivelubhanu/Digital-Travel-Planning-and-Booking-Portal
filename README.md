# Digital-Travel-Planning-and-Booking-Portal_Feb_Batch-8_2026

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```
2. Install PDF generation library (required for booking confirmation download):
   ```bash
   npm install jspdf --save
   ```
3. Run the development server:
   ```bash
   npm start
   ```

Features added in this branch include login authentication, a full booking form with validation, a confirmation summary, and automatic PDF itinerary generation.
